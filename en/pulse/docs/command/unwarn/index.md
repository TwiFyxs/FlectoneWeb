# Комманда /unwarn
Путь `command.yml > unwarn`

## Пояснение
Комманда для того, чтобы снять предупреждение с игрока
![command unwarn](/commandunwarn.png)

## Редактирование
```yaml
<command.unwarn>
```

### По умолчанию
```yaml
unwarn:
  enable: true
  range: -2
  aliases:
    - "unwarn"
  destination:
    type: CHAT
  cooldown:
    enable: false
  sound:
    enable: false
```
