# Комманда /unmute
Путь `command.yml > unmute`

## Пояснение
Комманда для того, чтобы размутить игрока
![command unmute](/commandunmute.png)

## Редактирование
```yaml
<command.unmute>
```

### По умолчанию
```yaml
unmute:
  enable: true
  range: -2
  aliases:
    - "unmute"
  destination:
    type: CHAT
  cooldown:
    enable: false
  sound:
    enable: false
```
