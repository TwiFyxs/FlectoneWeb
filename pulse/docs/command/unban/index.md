# Комманда /unban
Путь `command.yml > unban`

## Пояснение
Комманда для того, чтобы разблокировать игрока
![command unban](/commandunban.png)

## Редактирование
```yaml
<command.unban>
```

### По умолчанию
```yaml
unban:
  enable: true
  range: -2
  aliases:
    - "unban"
    - "pardon"
  destination:
    type: CHAT
  cooldown:
    enable: false
  sound:
    enable: false
```
