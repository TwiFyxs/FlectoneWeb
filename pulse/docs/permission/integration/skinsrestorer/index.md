# SkinsRestorer
Путь `permission.yml > integration.skinsrestorer`

## Пояснение
Категория, отвечающая за настройку прав у интеграции SkinsRestorer

## Редактирование
```yaml
<permission.integration.skinsrestorer>
```

### По умолчанию
```yaml
skinsrestorer:
  name: "flectonepulse.module.integration.skinsrestorer"
  type: TRUE
```
